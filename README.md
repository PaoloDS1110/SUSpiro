Lo scopo di questo progetto è la creazione di un file PDF facilmente usufruibile (scaricabile) da parte di insegnanti e studenti di chitarra con impostazione jazzistica.

Il solo file PDF non è sufficiente allo studente autodidatta, perché il materiale presentato è molto succinto e necessita di un insegnante che ne spieghi il significato, chiaramente illustrando il tutto con esempi suonati alla chitarra.

---
Per creare il file PDF è sufficiente eseguire il sequente comando

        > typst compile .\fondamenti.typ temp\fondamenti.pdf
